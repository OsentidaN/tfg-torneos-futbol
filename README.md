# TFG - Aplicación de Torneos de Fútbol

## Stack Tecnológico

### Backend
- TypeScript
- Node.js + Express
- Prisma ORM
- PostgreSQL
- JWT Authentication

### Frontend
- React + TypeScript
- Tailwind CSS
- React Router
- Axios

## Arquitectura
Modelo híbrido con ETL:
- API Externa (API-Football) → Script ETL → Base de Datos Propia → API Backend → Frontend
